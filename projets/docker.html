Projet sur Docker


Ce projet a pour but de comprendre comment utiliser Docker. Il permet de créer un environnement de développement et de production et de faire des tests avec une base de données.

On peut ajouter, supprimer et afficher la liste des personnes rentrées dans la base de données.

La base de données est géré par PhpMyAdmin en MySql.
